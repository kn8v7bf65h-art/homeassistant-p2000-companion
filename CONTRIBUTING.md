# Contributing

Bedankt voor je interesse in P2000 Companion.

## Problemen melden

Maak een GitHub Issue aan en voeg waar mogelijk toe:

- de versie van Home Assistant;
- de versie van P2000 Companion;
- relevante logregels;
- een geanonimiseerd voorbeeld van de P2000-melding;
- de gebruikte feed en filters.

## Wijzigingen voorstellen

1. Fork de repository.
2. Maak een aparte branch.
3. Test de wijziging in Home Assistant.
4. Open een pull request met een duidelijke omschrijving.

Voer geen persoonlijke adressen, tokens of andere geheime gegevens in broncode of issues in.
